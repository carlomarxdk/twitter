Network Analysis

Using the mentions,retweets and replys of each account, a network was created (If I talk about the network here, I always talk about the largest connected component. While there were always some unconnected nodes, most of them ($>95\%$) were always in the largest component.)

Full Network

Figure 1: Color according to Languages. Node size scaled after $\text{log}(\text{followers})$ of each account, with a minimum node size of $5$. (Because of the network size, this uses WebGL backend of plotly, might therefore look ugly on high-res displays.)

There are different clusters, most of them form according to the language of the nodes. 2 very self contained clusters are the german (orange, upper middle) and the spanish cluster (turquoise, middle right). They stand in contrast to the russian cluster, which is very interconnected with a lot of english accounts. However, there is also a completely seperate english cluster forming on the lower right, without a lot of accounts from different languages.
